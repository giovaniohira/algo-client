# Changelog

All notable changes to this project are documented here.

## [Unreleased]

## [0.2.0] - 2026-07-10

### Added
- Home dashboard with greeting, progress ring, and quick search
- Continue solving hero card with draft resume
- Recent submissions list and performance panel with difficulty bars
- Submission activity heatmap from LeetCode API (`submissionCalendar`)
- Profile screen with solve stats, donut chart, and LeetCode badges
- Sidebar navigation (Home / Problems / Profile)
- Per-difficulty solve counts from LeetCode profile API
- App logo across UI, README, and Electron icon
- Open source governance files (CONTRIBUTING, SECURITY, CODE_OF_CONDUCT)
- Code signing support in release pipeline (optional, via GitHub secrets)

### Changed
- Premium dark design system (tokens, cards, hover polish, 76px sidebar)
- Multi-view app shell with dedicated home and profile screens

## [0.1.0] - 2026-07-09

### Added
- Electron desktop client with Monaco editor
- LeetCode sign-in via embedded browser
- Local problem catalog with auto-sync
- Run / Submit with structured output panel
- Per-problem draft persistence
- Prev/next problem navigation
- GitHub Actions CI and Windows release pipeline
